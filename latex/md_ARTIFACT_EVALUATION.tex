\begin{DoxyItemize}
\item You can try a given benchmark X using \begin{DoxyVerb}make run_benchmark_X
\end{DoxyVerb}
 For example, you can try warp\+\_\+affine using \begin{DoxyVerb}cd <path-to-tiramisu>/build
make run_benchmark_warp_affine
\end{DoxyVerb}
 This will generate code from the Tiramisu warp\+\_\+affine benchmark and will run it. Code for this benchmarks is available in \href{https://github.com/Tiramisu-Compiler/tiramisu/blob/master/benchmarks/halide/warp_affine_tiramisu.cpp}{\texttt{ https\+://github.\+com/\+Tiramisu-\/\+Compiler/tiramisu/blob/master/benchmarks/halide/warp\+\_\+affine\+\_\+tiramisu.\+cpp}}
\item To try the cvtcolor benchmark \begin{DoxyVerb}make run_benchmark_cvt_color
\end{DoxyVerb}
 If you compile Tiramisu yourself, you need to make sure that libpng and libjpeg are installed and enabled in the configure.\+cmake file. Please refer to the \href{https://github.com/Tiramisu-Compiler/tiramisu/blob/master/benchmarks/README.md}{\texttt{ Benchmarks README}} file for more details. 
\end{DoxyItemize}
