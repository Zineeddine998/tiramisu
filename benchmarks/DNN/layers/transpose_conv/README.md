The files in this folder are organized as follows:

    General
        clean.sh : remove last build & run files.
        compile_and_run_mkldnn.sh : compile MKL-DNN code and run it.
        configure.h: define configuration constants.

    Tiramisu
        transpose_conv_generator_tiramisu.cpp: Tiramisu code generator.

    Wrapper
        transpose_conv_wrapper_tiramisu.cpp: wrapper file that calls the code generated by Tiramisu.

    Intel MKL-DNN
        transpose_conv_generator_mkldnn.cpp : code that calls Intel MKL-DNN Deconvolution.

To run this benchmark:

    At the directory build/benchmarks/DNN/layers/transpose_conv execute
	    make

    transpose_conv_wrapper_tiramisu executable will be created in the benchmark's directory.

    To compare the result of tiramisu with MKL-DNN execute :
        ./transpose_conv_wrapper_mkldnn.sh
    then
        ./transpose_conv_wrapper_tiramisu

    Execution results can be found in the text files : (if SAVE_TO_FILE_AND_COMPARE is set to 1 in the configure.h file)
        mkldnn_result.txt
        tiramisu_result.txt
